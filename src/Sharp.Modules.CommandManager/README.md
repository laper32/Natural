# CommandManager

## 简介

CommandManager 是一个提供集中式指令注册管理的插件。通过使用该插件，你可以清楚地追踪所有注册的指令来源，从而便于进一步的管理和处理。

## 主要特点

### 1. 集中式指令注册管理

该插件提供的是集中式的指令注册管理，换句话说如果其他地方注册了指令，你能知道是谁注册的，从而做进一步的处理。这种透明度使得指令管理更加可控和可追溯。

### 2. 多插件场景下的最佳选择

本插件在多插件的情况下会很实用，如果你不需要多插件而是大单体模式，Extension包依旧适合你。CommandManager 特别适合需要在多个独立插件之间协调和管理指令的场景。

### 3. 避免与 Extension 混用

**重要提示：** 不推荐Extension和该插件混用，混用会导致意想不到的情况。请根据你的项目架构选择其中一种方案：
- 多插件架构：使用 Sharp.Modules.CommandManager
- 单体架构：使用 Sharp.Extensions.CommandMangaer

## 使用场景

- ✅ 多插件项目，需要统一管理各插件注册的指令
- ✅ 需要追踪指令来源和注册者的项目
- ✅ 需要对指令进行集中化管理和控制的场景
- ❌ 简单的单体应用（推荐使用 Extension）
